# techBoyz

## Setup environment:

### Github steps (use GIT Bash) -><br/>
```
    1. git clone https://https://github.com/bogdanAntonASE/techBoyz.git
    2. cd techBoyz
```    
### FrontEnd -><br/>
```
    1. cd frontend
    2. npm install
    3. npm start
```    
### Backend -><br/>
```
    1. cd backend
    2. npm install
    3. npm start
```

Enjoy!
