# techBoyz

Setup environment:

Github steps ->
    1. open GIT Bash<br/>
    2. clone repository on your local machine using git clone https://https://github.com/bogdanAntonASE/techBoyz.git
    2. cd techBoyz
    
FrontEnd ->
    1. cd frontend
    2. npm install
    3. npm start
    
Backend ->
    1. cd backend
    2. npm install
    3. npm start
    
Enjoy!
